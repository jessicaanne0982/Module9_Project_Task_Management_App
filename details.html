The Tasks

Household Chores & Cleaning

  • Create recurring cleaning tasks (e.g., vacuuming, dusting, bathroom cleaning) with reminders for daily, weekly, or monthly schedules.
  • Create detailed checklists for each room (kitchen, bathroom, bedrooms, etc.)
  • Assign chores to specific family members or housemates with reminders.

Create a Household Chore Task
Meal Planning & Cooking

  • Plan your meals for the week, creating shopping lists based on the menu.
  • Create grocery shopping lists based on the meal plan. Add or remove items as needed.
  • Store favorite recipes in one location

Create a Meal Planning Task
Grocery Shopping & Supplies

  • Create grocery shopping lists based on needs (produce, dairy, household supplies) with reminders to restock.
  • Create lists for all the stores you shop

Create a Grocery Shopping Task
Home Maintenence & Repairs

  • Set reminders for routine home maintenance tasks (e.g., air filter replacement, HVAC system check, lawn care).
  • Create a log for home repairs, including a list of issues, tasks to fix, and the people responsible for them (plumber, electrician, etc.).
  • Have a calendar that highlights home maintenance tasks based on the season (e.g., winterizing pipes, spring cleaning).

Create a Home Maintenence Task
Organizing & Decluttering

  • Create specific decluttering tasks for different rooms or areas (e.g., closets, garage, attic).
  • Track items that need to be donated, including a reminder to take them to the donation center.

Create an Organization Task
Pet Care

  • Create reminders for feeding, grooming, and exercise for pets.
  • Keep track of veterinary appointments, vaccinations, and pet health needs.
  • Track pet supplies (food, toys, medications) and set reminders to be notified when they need to be replenished.

Create a Pet Care Task
Household Budget & Financial Management

  • Track household expenses (e.g., utilities, groceries, entertainment) and categorize them for easy budgeting.
  • Set up reminders for utility bills, rent, or mortgage payments, along with deadlines.
  • Track ongoing subscriptions (e.g., streaming services, home cleaning services) and their renewal dates.

Create a Household Budget Task
Household Documentation & Organization

  • Store and organize key household documents (e.g., warranties, insurance, lease agreements, medical records).
  • Keep a list of important contacts for services like plumbing, electricity, or pest control, along with emergency contact details.
  • Break down larger household projects into actionable tasks, with detailed steps for completion.

Create a Household Documentation Task
